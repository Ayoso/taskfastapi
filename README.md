Сервис управления документами (DMS)
Проект представляет собой минимальный сервис для работы с документами, реализованный на FastAPI и SQLAlchemy.

Выполненные требования: Реализованы все функциональные требования: API для загрузки файлов, автоматическое версионирование, API для получения актуального списка последних версий, API для базового AI-анализа с сохранением результата в БД. Добавлен маршрут для скачивания конкретной версии файла.

Технологический стек и запуск: Использованы Python, FastAPI, SQLAlchemy (SQLite). Проект полностью упакован с использованием Docker и Docker Compose.

Запуск: Для запуска проекта необходимо установить Docker Desktop и выполнить в корневой папке: docker-compose up --build -d.

Документация: После запуска API доступен по адресу http://127.0.0.1:8000/docs.
